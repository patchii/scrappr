# Google Play Store Review Extractor


Extract/Scrape Google Play Store Reviews of any Play Store Android Application and writes it in a csv file for further Data Exploration!

Essentials:

* Python
* Selenium Module in Python
* Chrome Binary
* Chrome Webdriver
